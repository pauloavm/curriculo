## ⚙️ Como executar este projeto localmente

Este projeto é estático e utiliza apenas HTML, CSS e JavaScript Vanilla. Para executá-lo em sua máquina:

1. Faça o clone do repositório:
   ```bash
   git clone [https://github.com/pauloavm/curriculo.git](https://github.com/pauloavm/curriculo.git)


2. Navegue até a pasta do projeto:
   ```bash
   cd curriculo

   ```


3. Abra o arquivo `index.html` em seu navegador de preferência ou utilize a extensão **Live Server** no VS Code.
